To use the program:

Step 1. Windows + R -> cmd
Step 2. cd "path to the project src folder", e.g. cd "C:\Workspace BU\InformationRetrieval-ADD\src"
Step 3. javac ParseTester.java
Step 4. java ParseTester

Now you should be prompted for the folder for files..
Enter, eg. C:\temp\test

Program outputs the inverted index

Now you should be prompted for the query words
Enter each word to search for separated by a space, eg. "Michael Jackson"

Program outputs document's in descending order in respect to their ranking
